# RealTime Room - WebSocket & WebRTC

RealTime Room is a real-time communication platform enabling 📹 video calls, 🎙️ audio, and 💬 text transfer within shared rooms. It utilizes **WebSocket** for instant messaging and room management, and **WebRTC** for peer-to-peer video and audio connections. 🚀 This project will continue to evolve with more features and details added.
